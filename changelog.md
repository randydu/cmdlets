Changelog
=========

0.4.1: ( Apr. 10, 2018)

* supports bash friendly cmdlet param delimiters.

  for bash, the '(', ')' is special characters so instead of add(1,2) we have to input "add(1,2)".
  now we can use add[1,2] as a workaround.
